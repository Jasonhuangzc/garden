## 🛠️ 后台管理面板升级 (Admin Panel)

**您提到的需求**: "后台加一个清空今日数据的功能... 积分要重新换，花要重新种... 简单美化下后台UI"

已完成以下升级：

### 1. 全新 UI 设计
- 移除了所有杂乱的 JSON 日志和不必要的调试按钮。
- **Status Dashboard**: 简洁显示上一次重置时间和当前花园状态 (已种/总数)。
- **Test Controls**: 核心区域，只有一个醒目的红色按钮。

### 2. "强力重置" 模式 (Hard Reset)
- 修改了 `admin.html` 按钮触发的逻辑，调用新的 `resetDailyData`。
- **效果**:
  - **金币 & 花园**: 全部清空。
  - **关键修改**: 将您的 `lastSyncedWords` (同步锚点) **强制归零**。
  - **测试场景**: 假设您今天已经背了 44 个单词。点击重置后，系统会“忘掉”这 44 个单词是已经领过的。下次同步时，它会惊喜地发现：“哇！您新背了 44 个单词！”，然后允许您再次领取 220 积分。
  - **目的**: 完美模拟“新的一天开始”的状态，让您可以反复测试“领积分 -> 买花 -> 种花”的完整闭环。

## 🚀 如何使用

1. 浏览器访问: `http://localhost:8000/admin.html`
2. 点击卡片中的 **[⚠️ Reset Daily Data (Fresh Start)]** 按钮。
3. 确认弹窗。
4. 看到绿色 "Success" 提示后，回到游戏主页 (`index.html`)。
5. 刷新游戏页面，您会发现：
   - 积分按钮变亮了 (Reward)
   - 金币归零了
   - 花园空了
6. 尽情测试吧！
